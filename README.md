## Zoom Image

A little yet powerful library to zoom image on wheel / hover / pinch actions. It is typical experience on e-commerce
sites. Examples are written with Preact, React, Svelte, Vanilla JS and Vue.

- ✅ Zoom on hover
- ✅ Zoom on wheel

## Development

```bash
git clone https://github.com/willnguyen1312/zoom-image
cd zoom-image
pnpm install
```

## Commands

### With Preact

```bash
pnpm start-preact
```

### With React

```bash
pnpm start-react
```

### With Svelte

```bash
pnpm start-svelte
```

### With Vanilla JS

```bash
pnpm start-vanilla
```

### With Vue

```bash
pnpm start-vue
```

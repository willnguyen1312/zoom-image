# Zoom Image

A little library to zoom image on hover

## Architecture

## Install

```bash
git clone https://github.com/willnguyen1312/zoom-image
cd zoom-image
pnpm install
```

## Developement

### With Vanilla JS

```bash
pnpm start-vanilla
```

### With Vue

```bash
pnpm start-vue
```
